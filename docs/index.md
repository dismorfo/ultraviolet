# Welcome to the NYU UltraViolet Docs Wiki

UltraViolet is an NYU repository application for research data. The codebase for UltraViolet is a local deployment of the [InvenioRDM](https://inveniosoftware.org/products/rdm/) application. NYU Libraries contributes to InvenioRDM as part of an open-source community.Â
